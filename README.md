# Página Nave Landing

Esta es una réplica de la página de landing de Nave.

## Archivos

- `index.html` - Estructura HTML de la página
- `styles.css` - Estilos CSS
- `nave.png` - Logo de Nave (debes agregar este archivo)

## Instrucciones

1. Coloca el archivo `nave.png` en esta carpeta
2. Abre `index.html` en tu navegador

## Nota

La imagen de la persona en el hero es un placeholder. Si tienes la imagen real, puedes reemplazarla en el CSS o HTML.
